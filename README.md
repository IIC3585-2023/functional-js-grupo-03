# Actividad Javascript Funcional

## Instrucciones

La actividad consisten en la transformación de archivo de texto en otro archivo de texto formateado de acuerdo a diferentes especificaciones.

* Cada frase debe comenzar con _n_ espacios en blanco (después de un punto seguido).
* Cada párrafo debe estar separado por _n_ líneas (después de un punto aparte).
* El ancho del texto debe ser a lo más _n_ (sin cortar palabras).
* Cada párrafo debe tener _n_ espacios de sangría.
* Se ignoran los párrafos que tienen menos de _n_ frases.
* Se ignoran los párrafos que tienen más de _n_ frases.
* Cada frase debe aparecer en párrafo aparte.
* Solo las primeras _n_ frases de cada párrafo.

Estas transformaciones deben poder combinarse o encadenarse de cualquier forma.
Es importante mantenerse dentro del **paradigma funcional** (idealmente usar solo funciones puras)
